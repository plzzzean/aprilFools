# Wedding Invitation Page

이 프로젝트는 React + Vite로 만든 모바일 청첩장 페이지입니다.

현재 페이지는 다음 내용을 포함합니다.

- 풀스크린 히어로 커버 이미지
- 신랑/신부 소개 이미지
- 예식 일시 및 장소 안내
- 오시는 길 지도 이미지와 네이버맵 링크
- 자동 페이드 갤러리
- 축의금 계좌 안내 및 계좌번호 복사 기능
- 카카오톡 공유를 위한 Open Graph 메타 정보

또한 이 페이지는 `april fool`에 사용되는 페이지입니다. (메롱)

## Run

개발 서버 실행:

```bash
npm run dev
```

프로덕션 빌드:

```bash
npm run build
```

빌드 결과 미리보기:

```bash
npm run preview
```

## Stack

- React
- Vite
- Tailwind CSS v4
